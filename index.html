NLP Causal Inference Network Analysis

How Twitter Influencers Move Housing Markets

Proved that influencer sentiment on Twitter causally shifts follower opinions and downstream housing prices, using instrumental variables on 1.2M real-estate tweets across 50 cities. Built graph-based network models to map the unidirectional flow of sentiment from influencers to followers.
